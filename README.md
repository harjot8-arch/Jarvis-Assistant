# 🤖 Jarvis - Voice-Controlled Python Assistant

**Jarvis** is a simple voice-activated personal assistant built in Python. It listens for the keyword "Jarvis", then responds to voice commands like checking the time, fetching weather, opening websites, and more.

## ✨ Features

- 🔊 Wake-word activation ("Jarvis")
- ⏰ Tells the current time
- 🌦️ Fetches live weather using OpenWeatherMap API
- 🌐 Opens common websites like Google, YouTube, GitHub
- 🛑 Listens for "stop" command to exit cleanly
- 🎙️ Text-to-speech feedback

## 🖥️ Tech Stack

- **Python 3.x**
- `speech_recognition` – for voice input
- `pyttsx3` – for speech output
- `requests` – for fetching weather data
- `webbrowser` – to open web pages
